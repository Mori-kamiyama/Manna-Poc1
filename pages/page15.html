最大値を返す関数を作る

次のプログラムは、ユーザーから2つの整数を入力し、大きい方の値を表示します。このプログラムを完成させるために、2つの整数を引数として受け取り、大きい方の整数を返す関数 max を作成してください。 #include // ここにmax関数を定義してください int main(void) { int x, y; printf("x="); scanf("%d", &x); printf("y="); scanf("%d", &y); int max_main = max(x, y); printf("%d\n", max_main); return 0; }

実行例

Input1
->x = 2
->y = 4

=>

Output1
4
Input2
->x=15
->y=8

=>

Output2
15

回答


int max(int x, int y)
{
    int max = 0; // 最大値を格納する変数
    if (x > y) {
        max = x; // xがyより大きい場合、xを最大値とする
    } else if (y > x) {
        max = y; // yがxより大きい場合、yを最大値とする
    }
    return max; // 最大値を返す
}
