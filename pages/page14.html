十進数->二進数

10進数を入力すると、二進数を返すプログラムを書いてください

実行例

Input1
->10

=>

Output1
1010
Input2
->974

=>

Output2
1111001110

回答


#include 
int main(void)
{
char two[9];
int ten = 0;
two[8]='\0';
printf("入力=");
scanf("%d", &ten);
for(int i = 7;i >= 0; i--){
    switch(ten % 2){
        case 1:
        two[i] = '1';
        break;
        case 0:
        two[i] = '0';
        break;
    }
    ten /= 2;
}
printf("%s", two);
return 0;
}
