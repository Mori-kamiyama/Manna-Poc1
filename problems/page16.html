四捨五入を返す関数を作る

次のプログラムは、整数を受け取り、その数を四捨五入して返すプログラムです。このプログラムを完成させるために、少数の入力を四捨五入して整数で返す関数 roundint_off を作成してください。 

                
#include 

// ここにmax関数を定義してください

int main(void)
{
float num;
printf("num=");
scanf("%f", &num);
printf("%d\n", roundint_off(num));
return 0;
}

実行例

Input1
->23.78

=>

Output1
24
Input2
->0.24

=>

Output2
0

回答


int roundint_off(float num)
{
    int roundint = num + 0.5;
    return roundint;
}
