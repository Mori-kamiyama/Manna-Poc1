5回Helloと言おう

5回「hello」と言ってください

実行例

Input1

=>

Output1
Hello
Hello
Hello
Hello
Hello

回答


#include <stdio.h>
int main(void)
{
    for ( i = 0; i < 5; i++ ){ //for文を使って特定回数繰り返す
        printf("Hello\n");
    }
return 0;
}

解説

このコードでのポイントはfor文です。
ここでは、while文との違いがポイントになってきます。 


//for文
for ( i = 0; i < 5; i++ ){
    //繰り返すコード
}

//while文
int i = 0;
while ( i < 5 ){
    //繰り返すコード
    i++;
}

このコードは同じく5回繰り返すコードです。しかしfor文の方がスマートであることがわかると思います。

for文はループの初期化、条件判定、更新を一行でまとめて書けるため、コードが見やすく整理されているのが特徴です。
特に、ループの繰り返し回数が決まっている場合や、カウンタ変数を用いた繰り返し処理に向いています。
これに対してwhile文は、初期化や更新をループ外で行う必要があるため、コードは煩雑化しやすく、ミスの原因となることもあります。

この問題はまだ簡単ですが、こういったミスをなくす癖をなくすために、for文を用いて問題を解きましょう。
