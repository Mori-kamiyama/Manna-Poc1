No.003

x回Helloと言おう

繰り返す回数を入力から受け取って、その回数分「Hello」と言ってください

実行例

Input1
->2

=>

Output1
Hello
Hello
Input2
->3

=>

Output2
Hello
Hello
Hello

回答


#include 

int main() {
    int count;  // 繰り返す回数を格納する変数

    printf("-> ");
    scanf("%d", &count);  // ユーザーから繰り返す回数を入力してもらう

    // 繰り返し処理を行う
    for(int i = 0; i < count; i++) {
        printf("Hello\n");
    }

    return 0;
}

解説

この問題のポイントは、scanf関数によって変数 count に値を代入し、その回数分繰り返すことです。

このscanf( "%d", &count );という並びで注目して欲しいのが、"%d"と&countです。

この場合、変換指定子を"%d"としているので、入力を整数として扱います。
これによりfor文で条件にcountをつかっても整数として正しく認識され繰り返すことができます。

次に、アドレス指定子を&countとしています。
このアドレス指定子はどの変数に値を入れるかを指定していて、＆の後の変数、今回はcountに格納されます。

この二つを正しく使うことで、「繰り返す回数を入力から受け取って、その回数分「Hello」と言う」ことができます。
