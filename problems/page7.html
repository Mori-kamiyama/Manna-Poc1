FizzBazz

口遊びをしましょう。100まで数のうち、２の倍数ならfizz,3の倍数ならbazz,6の倍数ならfizzbazzといい、それ以外の時はその時の番号を言います。
1から100までの数字を順番に数えるとき、以下のルールに従って言い換えてください。

• 数字が2の倍数なら「fizz」と言う。
• 数字が3の倍数なら「bazz」と言う。
• 数字が6の倍数なら「fizzbazz」と言う。
• それ以外の数字は、そのまま数字を言う。

このルールにしたがって1から100まで出力してみてください

実行例

Input1

=>

Output1
1
fizz
bazz
fizz
5
fizzbazz
〜

回答


#include 
int main(void)
{
    for (int i = 1;i <= 100; i++){
        printf("%d", i);
        if ( i % 2 == 0){
            printf("Fizz");
        }
        if ( i % 3 == 0){
            printf("Bazz");
        }
        if ( i % 2 != 0 && i % 3 != 0 ){
            printf("%d", i);
        }
            printf("\n");
        }
    return 0;
}
