文字列の長さを測る

文字列を受け取って、その長さを測ってください。

実行例

Input1
->Hello World

=>

Output1
11
Input2
->I using Manna

=>

Output2
13

回答


#include 
int main(void)
{
    char word[50];
    int i = 0;
    int count = 0;

    printf("入力=");
    scanf("%s", word);
    while(word[i] !='\0'){
       count ++;
       i++;
    }

    printf("文字列:%d\n", count);
    return 0;
}
