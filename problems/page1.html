100まで足し続ける

入力された数を足していって、100を超えたら、「100を超えました」と表示してください。

実行例

Input1
->10
->20
->70

=>

Output1
100を超えました。
Input2
->90
->20

=>

Output2
100を超えました。

回答


#include 

int main() {
    int sum = 0;  // 合計値を初期化
    int input;    // 入力される数値を格納する変数

    // ループを使って数を足していく
    while (sum <= 100) {
        printf("-> ");
        scanf("%d", &input);  // ユーザからの入力を取得
        sum += input;         // 入力された数を合計に加算

        // 100を超えたらメッセージを表示
        if (sum > 100) {
            printf("100を超えました。\n");
        }
    }

    return 0;
}

解説

ここでポイントなのは　sum = 0;　です。

変数を定義するだけでなく、初期化することで正しく合計値を測ることができます。
変数は内部的にはこのように保存されています。

[1][0][1]...[0][1][1]

このとき変数を定義しただけだと、あなたのパソコンのメモリにこのように場所が割り当てられるだけです。
メモリはプログラミング以外にもゲームやアプリケーションの実行で使われるので場所が割り当てられても前に使ったデータがそのままになっていることがあります。
移動教室で一つ前の人が机に本やゴミを置きっぱなしにしているようなものです。
そのため一度初期化することで正しく足していくことができます。

次に、whileを使って初期化されたsumが100を超えるまで繰り返します。


while (sum <= 100) {
    printf("-> ");
    scanf("%d", &input);  // ユーザからの入力を取得
    sum += input;         // 入力された数を合計に加算

    // 100を超えたらメッセージを表示
    if (sum > 100) {
        printf("100を超えました。\n");
    }
}

これで入力された数を足していって、100を超えたら、「100を超えました」と表示することができます。
