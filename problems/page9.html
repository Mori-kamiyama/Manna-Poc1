配列を逆転する

配列の長さと要素を受け取って、配列を左右反転させるコードを書いてください。

実行例

Input1
->配列の長さ = 5
->配列の要素 = 1 2 3 4 5

=>

Output1
5 4 3 2 1

回答


#include 
int main(void)
{
    //配列を受け取る
    int nagasa = 0;
    printf("配列の長さ=");
    scanf("%d", &nagasa);
    int num[nagasa];
    printf("配列の要素=");
    for (int i = 0; i < nagasa; i++){
        scanf("%d", &num[i]);
    }

    //配列を逆向きに出力する
    for (int j = nagasa - 1 ; j >= 0; j--){
        printf("%d ", num[j]);
    }
    return 0;
}
