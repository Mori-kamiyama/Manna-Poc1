メニュー表を作る

レストランのメニューに対して、最初の1文字を入力すると、それに対応する価格が表示されるプログラムがあります。以下のメニューから1つ選び、その頭文字を入力してください：

• カレー (c): 500円
• ステーキ (s): 600円
• パスタ (p): 650円
• バーガー (b): 450円
• 水 (w): 0円

実行例

Input1
->c

=>

Output1
500円です
Input2
->w

=>

Output2
0円です

回答


#include 
int main(void)
{
    char str[20];
    int i = 0;
    printf("メニュー=");
    scanf("%s", str);

    switch(str[i]){
        case 'c':
        printf("500円です\n");
        break;
    case 's':
        printf("600円です\n");
        break;
    case 'p':
        printf("650円です\n");
        break;
    case 'b':
        printf("450円です\n");
        break;
    case 'w':
        printf("0円です\n");
        break;
    default:
        break;
    }
    return 0;
}
